如何实现在editplus下运行jar文件里的内容呢？
DEMO文件夹下面有一个Animal.jar 然后有一个测试文件Demo.class
Demo 调用 直接 把jar里面的文件视为当前目录下的结构就行
如jar文件的架构如下：
jar(bwn.Animal类 bwn为包名 bwn.Cat类)
Demo.class内容如下
import bwn.*;
public class Demo {
public static void main(String[] args) {

System.out.println(“Hello World!”);

Animal a=new Cat();
a.eat();
a.sleep();

}
}
cmd进入DEMO目录下
然后设置环境变量
set classpath=Animal.jar;.
不加上;或者不加上;.那么只能通过编译无法运行，如果命令都不敲
就没法编译
editplus里的自定义工具里不支持set命令 所以只能通过editplus
