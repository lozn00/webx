MorphVOX Pro

门槛低，质量差
要在游戏/QQ中生效，必须给游戏软件选择麦克风设备为Screaming Bee Audio设备

改软件支持插件加载，插件又有vst插件，vst插件有有支持vsdt插件的 morphvox插件。

图中有

文本转语音插件 、测试朗读插件、vst effect插件。 dj stream插件
变音有自带的也可以通过proph vox effect rack里面的dll插件来控制

绿色代表声音通过此插件处理。
插件安装方法

C:\Users\lozn\AppData\Roaming\Screaming Bee\Plugins\Effects Rack Plugin\VST Plugins

Samplitude Pro

下图是我设置的效果，所有地方都需要设置，错了一个都没法变音，或者变音只能自慰

我不知道是不是必须开启监听才能变音，但是总感觉监听那么效果肯定不理想，反正是一个死胡同，不监听就听不到变音，监听模式中的第4和第6个都能听到变音效果
aiso设备选择aiso link pro 然后设备列表选择自己的麦克风设备和自己的想听到变音的设备，我这里都选择耳机，而window设置里面我设置的是mix01,另外还有一个地方是输出选择的1-2，因为我工具-音轨也是这样设置的，不然也是听不到声音的
另外还需要把御姐的那个喇叭点成蓝色才有效果。。

总之门槛比较高，我搞了几天了，搞郁闷了，网上的教程压根没提到怎么在其它软件使用，好像自然而然就能用了，更加没有说到监听模式对其它软件的影响，以及不同aiso驱动对其它软件是否能录变音的结果。这让我一个小白情何以堪，总之我搞那么久还是没搞懂，这教程纯属一个笔记。。
至于voicemetter ，b1是输入设备a1是输出设备，其中 有一些滑块是控制电脑中软件的声音是否被录入。

只能自己听到变音，效果还不错，而且这个又分为虚拟跳线等设置，而虚拟跳线软件又有好几种，和自带声卡的那种(可能这种就比较容易设置)我这种声卡耳机都只能用虚拟跳线了，。

而虚拟跳线软件又有三种，每一种使用方式不同，我特意买了一个usb声卡耳机，但是还是没有这种asio驱动只能用软件的跳线软件，但是这些软件，其中asio link pro我成功让qq听到了变音，但是必须设置监听模式才行，也就是滑动条弄到最右边，但是这显然不是最好的方式，效果也非常烂，另外它会占用被使用的设备，如果其他软件把设备设置为它，或者window中测试播放此设备会提示占用。。。。
而Voicemetter我是怎么都没法给其他软件出效果，头大了！

跳线软件中的设置

改图中，最左边的硬件hareware input1选择麦克风录入设备，我这里是usb声卡耳机，所以选择这个，后面2个硬件没必要了，不需要同时多个录入设备。
然后打开hareware input1的B2,我也不知道为什么b1不行，。。。
然后第4个 ，代表的是其它软件的录入拦截， 开启b2就把麦克风变音的声音录入，
开启A1就把电脑里面的一些软件播放的声音也给录入给第三方软件。。。
第5个， 是输出。勾选A1,另外右上角的A1,A2,A3中A1我选择的是自己的耳机输出设备，所以能听到自己说的话以及变音。
点开A1可能有很多，这里可以举一反三，MME包含了voicemetter 所以我选择的是MME里面的usb声音设备，而不是网上说的WDM..
