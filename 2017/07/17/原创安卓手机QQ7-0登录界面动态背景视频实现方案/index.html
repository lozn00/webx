总结:

由于加载视频比较慢，比如会有黑屏，但是图片的加载速度所产生的黑屏人几乎感受不到,所以采用的方法是 先获取第一张视频第一帧图片 覆盖在视频的区域上面，
当视频加载回调首帧的时候隐藏，
